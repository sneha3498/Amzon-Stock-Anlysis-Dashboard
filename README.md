# Amzon-Stock-Anlysis-Dashboard
An interactive and dynamic data analysis project designed to track, clean, and visualize Amazon's historical stock performance metrics from 2012 to 2025 using Advanced Excel.

### 🎯 Project Objective

The main goal of this project was to transform raw, unstructured financial datasets into a functional visual dashboard. This helps analyze market volatility, evaluate trading volumes, and spot long-term growth trends for data-driven financial decision-making.

### ⚙️ Data Analytics Process & Methodology

To build this dashboard from scratch, the following end-to-end data pipelines were executed:

1.  Data Cleaning & Preprocessing:
    *   Handled missing values and removed duplicate records from the raw dataset.
    *   Formatted and structured date-time columns to enable chronological filtering.
2.  Exploratory Data Analysis (EDA):
    *   Derived statistical summaries including the **Moving Averages** of Open, Close, High, and Low stock prices.
    *   Aggregated monthly and yearly trends to understand seasonal market behavior.
3.  Data Modeling & Pivot Tables:
    *   Structured relational pivot data to calculate complex fields like *Trading Volume Analysis* and *High VS Low by Day*.
4.  Dashboard Architecture & UI Design:
    *   Designed a clean, professional user interface with high-contrast color coding for critical financial KPIs.
    *   Implemented dynamic, multi-criteria cross-functional **Slicers (Days, Months, Years)** for granular data filtering.

### 📈 Key Insights Captured

*   Volume Distribution: Tracked trading volumes across different days of the week to analyze market activity patterns.
*   Price Growth: Visualized long-term yearly adjusted close prices, capturing Amazon's major growth cycles up to 2025.
*   Price Volatility: Monitored the direct relationship between Open/High and High/Low prices on a daily baseline.

### 🛠️ Tech Stack & Tools Used

*   Advanced Excel (Pivot Tables, Advanced Formulas, Data Aggregation)
*   Data Visualization (Dynamic Slicers, Timeline Filters, Interactive Line/Bar/Pie Charting)
*   Data Management (Data Cleaning, Filtering, and Structured Data Pipelines)
